# Retrievy – Lost & Found Platform

Retrievy is a full-stack Lost & Found web application built using Python and SQLite, designed to help users report, search, and recover lost items efficiently. The app supports item listings, image uploads as of now.

---

##  Features

- Report lost or found items
-  Filter items by category, location, and type
-  Upload and store item images
-  SQLite database integration
-  Clean UI (Streamlit-based interface)
-  Lightweight and easy to deploy

## Upcoming Features

-  User authentication (secure login/signup using bcrypt)
-  Search items by category, location, and type
-  Basic matching logic to connect lost & found reports
-  SQLite database integration

---

## Tech Stack

- Python 
- SQLite 
- Streamlit (frontend UI)
- OS / file handling for image storage
